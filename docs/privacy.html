개인정보 처리방침

오늘달은 사용자의 개인정보를 수집하거나 외부 서버로 전송하지 않습니다. 계정 가입, 광고, 사용자 추적 및 분석 도구를 사용하지 않습니다.

위치 정보

위치 권한은 현재 위치를 기준으로 월출, 남중, 월몰 시각을 계산할 때만 사용합니다. 위치 정보는 기기 안에서 처리되며 저장하거나 개발자 또는 제3자에게 전송하지 않습니다. 권한을 허용하지 않아도 앱에서 지역을 직접 선택해 사용할 수 있습니다.

알림

알림 권한은 사용자가 달 알림 기능을 켤 때 요청합니다. 보름달, 삭, 상현·하현 및 월출 알림은 기기에서 로컬 알림으로 예약되며 외부 서버를 사용하지 않습니다.

기기 내 저장

선택한 지역, 시간 표시 방식, 달력 설정 및 알림 설정은 앱 사용을 위해 기기의 UserDefaults에 저장됩니다. 달 사진 공유 기능은 사용자가 공유할 파일을 기기의 임시 저장 공간에 만들며, 공유 대상은 사용자가 직접 선택합니다.

제3자 구성요소

오늘달은 천문 계산을 위해 오픈 소스 라이브러리 SwiftAA를 사용합니다. 계산은 기기 안에서 수행되며 해당 라이브러리를 통한 개인정보 전송은 없습니다.

아동의 개인정보

오늘달은 연령과 관계없이 개인정보를 수집하지 않습니다.
