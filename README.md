# 오늘달 iOS Prototype

SwiftUI iPhone prototype for the first visual direction: a moon-first Korean moon phase app.

## What is included

- Today tab with a large realistic moon image, brightness, moon age, visibility status, moonrise, transit, moonset, and next full moon CTA.
- Calendar tab with tappable July 2026 lunar days and a selected-day detail panel.
- Notifications tab with toggleable reminder preferences and permission-copy preview.
- Location and settings sheets.
- XCTest coverage for the core fixture copy and major lunar phase data.

## Open

Open `OneulDal.xcodeproj` in Xcode and run the `OneulDal` scheme on an iPhone simulator.
