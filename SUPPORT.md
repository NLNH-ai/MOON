# 오늘달 고객지원

오늘달 사용 중 문제가 발생했거나 개선 의견이 있다면 [오늘달 고객지원 페이지](https://nlnh-ai.github.io/MOON/support.html)를 확인해 주세요. 해결되지 않는 문제는 페이지의 GitHub 문의 링크를 통해 등록할 수 있습니다.

문의 시 아래 내용을 함께 적어 주시면 확인에 도움이 됩니다.

- 사용 중인 iPhone 모델과 iOS 버전
- 오늘달 앱 버전
- 문제가 발생한 화면과 재현 순서
- 가능한 경우 화면 캡처

## 자주 확인할 항목

- 월출·월몰 시간이 예상과 다른 경우 앱의 기준 위치를 확인해 주세요.
- 현재 위치를 사용하려면 iPhone 설정에서 오늘달의 위치 권한을 허용해 주세요.
- 알림이 오지 않으면 iPhone 설정에서 오늘달의 알림 권한과 앱 안의 알림 스위치를 확인해 주세요.

개인정보 처리에 관한 내용은 [공개 개인정보 처리방침](https://nlnh-ai.github.io/MOON/privacy.html)에서 확인할 수 있습니다.
